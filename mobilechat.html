Introduction

Welcome to The Chatroom demo. The messages are stored in a text file on the server. Every three seconds the the chat file is checked for updates, if it was recently updated, it is reprinted. You will be prompted for a nickname. Php scripts only allow a maximum of 200 lines at a time in the file. For some reason the timestamps are misaligned in firefox. Enjoy!
